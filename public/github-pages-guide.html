🚀 تبادل - دليل النشر

انشر تطبيقك على GitHub Pages في 5 دقائق

⚡ سريع | 🆓 مجاني | 🌍 متاح للعالم

الخطوة 1: بناء المشروع

افتح Terminal في مجلد المشروع واكتب:

npm run build

✅ يجب أن ترى رسالة "built in X.XXs"

الخطوة 2: تهيئة Git

git init git add . git commit -m "إطلاق تطبيق تبادل"

الخطوة 3: إنشاء مستودع GitHub

  • اذهب إلى github.com/new
  • اسم المستودع: tabadul
  • اجعله Public
  • اضغط Create repository

الخطوة 4: ربط ورفع المشروع

استبدل YOUR_USERNAME باسمك على GitHub:

git remote add origin https://github.com/YOUR_USERNAME/tabadul.git git branch -M main git push -u origin main

الخطوة 5: تفعيل GitHub Pages

  • في مستودعك، اذهب إلى Settings
  • اضغط Pages من القائمة
  • Branch: main / Folder: / (root)
  • اضغط Save

🎉 انتهى!

خلال 1-3 دقائق، سيكون تطبيقك متاحاً على:

https://YOUR_USERNAME.github.io/tabadul/

استبدل YOUR_USERNAME باسم المستخدم الخاص بك!

اقرأ الملفات التالية في مجلد المشروع:

  • ⚡ QUICK_START.md - دليل سريع
  • 🎯 FINAL_DEPLOYMENT.md - دليل شامل
  • 📖 README.md - وصف المشروع
