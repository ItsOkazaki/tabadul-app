📋 قائمة التحقق المسبق

المشروع يعمل محلياً

🎯 الخطوة 1: بناء المشروع

افتح Terminal في مجلد المشروع واكتب:

npm run build
💡 نتيجة متوقعة: يجب أن ترى رسالة "✓ built in X.XXs"

🎯 الخطوة 2: تهيئة Git

git init git add . git commit -m "إطلاق تطبيق تبادل"

🎯 الخطوة 3: إنشاء مستودع GitHub

  • اذهب إلى github.com/new
  • اسم المستودع: tabadul (أو أي اسم تريده)
  • الوصف: "منصة تبادل المهارات بين الطلاب"
  • اجعله Public (عام)
  • لا تُضف README أو .gitignore أو LICENSE
  • اضغط Create repository

🎯 الخطوة 4: ربط ورفع المشروع

استبدل YOUR_USERNAME باسم المستخدم الخاص بك على GitHub:

git remote add origin https://github.com/YOUR_USERNAME/tabadul.git git branch -M main git push -u origin main
⚠️ مهم: استبدل YOUR_USERNAME باسمك الحقيقي على GitHub!

🎯 الخطوة 5: تفعيل GitHub Pages

  • في مستودعك على GitHub، اذهب إلى Settings (الإعدادات)
  • من القائمة الجانبية، اضغط Pages
  • تحت Build and deployment:
    • Source: اختر Deploy from a branch
    • Branch: اختر main
    • Folder: اختر / (root)
  • اضغط Save (حفظ)

🎉 تهانينا!

خلال 1-3 دقائق، سيكون تطبيقك متاحاً على:

https://YOUR_USERNAME.github.io/tabadul/

استبدل YOUR_USERNAME باسم المستخدم الخاص بك!

🐛 حل المشاكل الشائعة

الصفحة بيضاء تماماً

npm run build # تحقق من vite.config.ts يحتوي على: base: './'

خطأ 404

  • انتظر 2-3 دقائق بعد النشر
  • امسح كاش المتصفح: Ctrl+Shift+R
  • تحقق من Settings > Pages
